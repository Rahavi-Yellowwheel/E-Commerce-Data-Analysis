# E-Commerce-Data-Analysis
E-commerce data analysis involves examining online sales, customer behavior, product performance, and return patterns using data from orders, customers, products, and returns. The goal is to identify trends, measure business performance, and find insights that help improve sales, customer experience, and operational efficiency.
# Dataset Description
Orders.csv
Contains order-level details
Example columns: order_id, order_date, customer_id, product_id, unit price,quantity, total_amount,return_flag
Customers.csv
Contains customer demographic and profile details
Example columns: customer_id, customer_name, age_group, location, loyalty_segment
Products.csv
Contains product-level metadata
Example columns: Product_id, Product_name, Product_category,Brand
Returns.csv
Contains return order information
Example columns: return_id, order_id, return_reason, return_date
